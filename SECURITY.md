# Security Policy

## Sensitive data

Do not commit real API keys, Gemini cookies, `.env` files, generated workspace files, or rendered media outputs.

Sensitive local values include:

- `LLM_API_KEY`
- `IMAGE_API_KEY`
- `GEMINI_SECURE_1PSID`
- `GEMINI_SECURE_1PSIDTS`

If any credential has been committed or shared publicly, revoke or rotate it immediately.
